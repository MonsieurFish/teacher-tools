How to Export & Import Grades

D2L Exporting grades from D2L Brightspace
Your Course▸ Grades▸ Enter Grades▸ Export
  1. Open your course in Brightspace and click Grades in the navbar.
  2. Click Enter Grades, then click the Export button.
  3. Under Key Field, check OrgDefinedId (required for import to work).
  4. Under Grade Values, check Points Grade.
  5. Check the grade items you want, then click Export to CSV.
Tip: Always export as CSV (not Excel). Only CSV can be re-imported into D2L.
D2L Importing grades into D2L Brightspace
Your Course▸ Grades▸ Enter Grades▸ Import
  1. Go to Grades > Enter Grades and click Import.
  2. Choose the CSV you downloaded from this tool.
  3. Check "Create new grade item when unrecognized item is referenced" if the assignment doesn't exist yet.
  4. Click Import, review the preview, then confirm.
Important: D2L matches students by OrgDefinedId. This tool pulls the correct IDs from your roster export, so make sure you uploaded that in Step 2.
PowerSchool Exporting scores from PowerTeacher Pro
PowerTeacher Pro▸ Scoresheet▸ Gear icon▸ Export Scores
  1. Open PowerTeacher Pro and select your class.
  2. On the Scoresheet view, click the gear icon (top-right).
  3. Click Export Scores and save the CSV.
Note: If you don't see the Export option, your district admin may need to enable it.
PowerSchool Importing scores into PowerTeacher Pro
PowerTeacher Pro▸ Scoresheet▸ Gear icon▸ Import Scores
  1. Open PowerTeacher Pro, go to the Scoresheet for your class.
  2. Click the gear icon and select Import Scores.
  3. Select the CSV from this tool, verify the column mapping, then click Import.
Important: PowerSchool matches students by Student_Number. This tool pulls the correct IDs from your roster export.
Using the grading template

The downloadable template has three columns:

Last Name First Name Assignment Name
SmithJane95
DoeJohn82
  1. Rename the third column header from Assignment Name to your actual assignment name (e.g., Quiz 3 or Midterm Essay). This becomes the assignment name in the import file.
  2. Fill in student last names, first names, and grades.
  3. You can add extra columns for your own notes — the tool will ignore them.
  4. Do not rename the Last Name or First Name headers, or the tool won't recognize them.
  5. Save and upload to this tool as your grades file (Step 3).
The template also has an Instructions sheet tab with these same steps for quick reference.
