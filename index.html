How to Export & Import Grades

Step-by-step instructions for D2L Brightspace and PowerSchool. Click a section to expand it.

D2L Exporting grades from D2L Brightspace

To get your grades out of D2L so you can transfer them elsewhere:

Your Course ▸ Grades ▸ Enter Grades ▸ Export
  1. Open your course in Brightspace and click Grades in the navbar.
  2. Click the Enter Grades tab at the top.
  3. Click the Export button.
  4. On the Export page, configure your options:

  4. Click Export to CSV. A .csv file will download.
Tip: Always use CSV (not Excel) if you plan to re-import later. Only CSV supports round-trip import in D2L.
D2L Importing grades into D2L Brightspace

To bring grades into D2L from a CSV file (e.g., one generated by this tool):

Your Course ▸ Grades ▸ Enter Grades ▸ Import
  1. Open your course in Brightspace and go to Grades > Enter Grades.
  2. Click the Import button.
  3. Click Choose File and select your CSV.

  1. Optionally check "Create new grade item when unrecognized item is referenced" if the CSV contains assignment names that don't exist in your gradebook yet.
  2. Click Import. D2L will show a preview of changes.
  3. Review the preview carefully — check that students and scores look correct.
  4. Confirm the import.
Important: D2L matches students by the OrgDefinedId or Username column, not by name. If these columns are blank in your CSV, D2L won't know which student gets which grade. You may need to first export from D2L to get the student IDs, then use this tool to merge in the grades.

The CSV must end each row with a # in the "End-of-Line Indicator" column. This tool adds that automatically.

Recommended workflow: Export from D2L first (to get student IDs) → use this tool to merge in external grades → import the result back into D2L.
PowerSchool Exporting scores from PowerTeacher Pro

To export scores from PowerTeacher Pro's gradebook:

PowerTeacher Pro ▸ Scoresheet ▸ Menu (gear icon) ▸ Export
  1. Log into PowerSchool and open PowerTeacher Pro from the start page.
  2. Select the class/section you want to export from the left sidebar.
  3. Make sure you are on the Scoresheet view (the grid of students and assignments).
  4. Look for the gear icon or menu icon (usually in the top-right area of the scoresheet).

Recommended workflow for transferring D2L grades to PowerSchool:
1. Export from D2L (your graded assignments)
2. Export from PowerTeacher Pro (to get the student number template)

Don't see the Import option? Your district admin may need to enable it. Score import was added in PowerSchool version 20.x and must be turned on at the district level. Contact your school's IT department if the option is missing.

General tips & common issues

Student matching between platforms

D2L and PowerSchool use different student IDs. D2L uses "OrgDefinedId" and/or "Username," while PowerSchool uses "Student_Number." When transferring between them, this tool matches students by name. Double-check the results in the review table, especially for students with similar or identical names.

Name format differences

D2L exports separate "Last Name" and "First Name" columns. PowerSchool sometimes uses a combined "StudentName" column in "Last, First" format. This tool handles both automatically, but always review the parsed names in the table.

What if my file isn't recognized?

If the tool shows the "Identify Columns" step, it means the column headers didn't match a known format. Just use the dropdowns to tell the tool which column is the name, which are grades, etc. This happens with hand-made spreadsheets or non-standard exports.

Editing before export

You can click on any cell in the review table to edit it directly. Need to fix a name, adjust a score, or add a student ID? Just click and type. The exported CSV will reflect your edits.

Comments and feedback

D2L's CSV import does not support comments or feedback text — only numeric grades. PowerSchool's import has limited comment support in newer versions. If you need to import comments, you'll need to enter them manually or use the platform's API.
